cloudOS

cloudOS is a project I started a long time ago when I was bored. It is essentially ChromeOS, but not as functional, working only using an iframe. It's intended to be a very simplified version of the internet, however, many sites block using iframes, so many sites don't work and have to be opened in a new tab.

The Material You update for cloudOS is here, it still has a few bugs, so please report those on GitHub if possible.
